This is a sample project entry. To add your real projects, create new `.md` files in the `src/content/projects/` directory.

## Frontmatter

Each project file should have frontmatter like this:

```yaml
---
title: "My Project Name"
description: "A brief one-line description of the project."
tags: ["Docker", "Python", "Kubernetes"]
link: "https://example.com"
github: "https://github.com/ekelhala/project-repo"
featured: true
year: 2025
---
```

## Body Content

You can add more detailed descriptions in the markdown body. This will be useful if you create dedicated project pages in the future.
